🔗 Data Unification Tool

This project is a Python-based solution that normalizes and merges JSON datasets that use different timestamp formats into a single unified format.

📌 Project Overview

In real-world applications, data often comes from multiple sources and is represented in different formats.
This project demonstrates how to:

Convert timestamps from ISO 8601 format to milliseconds

Merge multiple JSON datasets into a standardized structure

Validate the output against an expected result

The final result is a unified dataset that can be used for analytics, monitoring, or further processing.

🎯 Objectives

Read data from multiple JSON files

Normalize timestamp formats

Merge datasets into one

Generate a unified output

Automatically verify correctness

Language: Python 3

json → for handling JSON data

datetime → for timestamp conversion
